# Java OOP Practice 🚀

## About
This repository contains Java OOP practice code
written during my BSCS 2nd semester at NUML.

## Topics Covered

### 1. Encapsulation
- Private fields with getters and setters
- Validation in setters
- Real-world examples: Person, BankAccount, Hospital System

### 2. Inheritance
- extends keyword and super()
- Method overriding with @Override
- Multilevel inheritance
- Real-world examples: Animal/Dog/GuideDog, Person/Student/ScholarshipStudent

## Projects
| Class | Concepts Used |
|---|---|
| Person | Encapsulation, Validation |
| BankAccount | Encapsulation, deposit/withdraw logic |
| Hospital | Object as a field |
| Animal/Dog | Single Inheritance |
| GuideDog | Multilevel Inheritance |
| ScholarshipStudent | Encapsulation + Inheritance combined |
